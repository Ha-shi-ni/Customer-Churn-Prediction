# Customer Churn Prediction & Business Analytics Dashboard

## 📌 Project Overview

This project predicts customer churn using Machine Learning and analyzes business impact using Power BI.

## 🎯 Objectives

* Predict customer churn probability
* Estimate revenue loss due to churn
* Perform what-if analysis for churn reduction
* Visualize insights using Power BI dashboard

## 🛠 Tools & Technologies

* Python (Pandas, Scikit-learn)
* Jupyter Notebook
* Power BI
* Excel

## 📊 Key Features

* Churn prediction model (Random Forest)
* Feature engineering and EDA
* Revenue impact analysis
* Interactive dashboard with KPIs

## 📈 Business Insights

* Identified high-risk customer segments
* Estimated potential revenue loss
* Suggested churn reduction strategies
